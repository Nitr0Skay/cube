# Simple Cude

## Purpose
It is simple cube which I created almost ten years ago. Now I want to get back to work with Three.js library and Improve my skills

## Features
- It can change color in two ways:
  1. You can select one from CSS pre defined colors
  2. You can just input RGB values separatetly
- You can change the size of it  
- You can change the rotation speed
- You can change the falling speed
- You can pause rotation or falling animations
